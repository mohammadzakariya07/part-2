# Smart Aadhaar Center Monitoring System (Prototype)

This project demonstrates a web-based authentication and monitoring
interface for Aadhaar centers. The system simulates secure login
and centralized monitoring of Aadhaar enrolment and update services.

## Features
- Operator-based secure login
- Central dashboard accessible from anywhere
- Monitoring of Aadhaar services
- IoT/ESP integration ready (future scope)

## Technology Used
- HTML
- CSS
- JavaScript
- GitHub Pages (Hosting)

## Disclaimer
This is a prototype developed for hackathon and educational purposes.
No real Aadhaar data is used.
